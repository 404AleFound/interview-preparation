## 项目简介

`interview-preparation` 是一个面向后端 / 系统 / 网络 / 数据库 面试准备的知识仓库，收集并整理了 C++ 语法、计算机网络、操作系统、MySQL 数据库、以及与 AI 代理相关的学习笔记与示例资源。

本仓库目标：
- 为准备技术面试的开发者提供系统化、可阅读的笔记与示例。 
- 作为复习材料、练习题与面试答案的索引库。

---

## 目录结构（概要）

仓库按大主题目录划分，主要内容如下：

- `1-cpp-grammer/`：C++ 语法与相关示例、模板（基础语法、重要特性总结）。
- `2-computer-network/`：计算机网络专题，包含多个子目录：
	- `base/`：网络基础概念（可能包含协议概览、模型等）。
	- `dns/`、`http/`、`ip/`、`ssl-tls/`：常见协议专题。
	- `tcp/`：TCP 深入笔记（例如：报文格式、三次握手、四次挥手、滑动窗口、拥塞控制、重传机制、socket 编程等），并附带 `assets/` 图片示意。
	- `udp/`、`webSocket/`：UDP 与 WebSocket 相关笔记（若有）。
- `3-operating-system/`：操作系统相关，包含 `基础/`、`进程管理/`、`内存管理/` 等子目录，用于记录进程、调度、内存分配与管理等主题。
- `4-database-mysql/`：MySQL 相关笔记，分为 `基础篇/`、`架构篇/`、`内存篇/`、`日志篇/`、`实践篇/`、`事务篇/`、`索引篇/` 等，包含执行流程、索引结构（B+Tree）、Explain 分析等资料，配有图片示例。
- `5-ai-agent/`：AI agent / 项目相关内容（可用于存放实验、示例、笔记）。

仓库内大部分文件为 Markdown 笔记和静态图片（位于各 `assets/` 文件夹中）。

## 如何使用本仓库

- 本仓库以笔记为主，直接在浏览器或编辑器中打开即可阅读（推荐使用 VS Code 或 GitHub 在线预览）。

本地快速查看示例：

```powershell
# 在仓库根目录打开 VS Code（Windows PowerShell）
code .
```

- 搜索：使用编辑器全局搜索关键词（如 "TCP"、"B+Tree"、"三次握手"）可快速定位相关笔记。
- 图片说明：每个子目录下的 `assets/` 文件夹包含示意图，配合对应 Markdown 阅读更直观。

建议的学习顺序（可按需调整）：
1. `1-cpp-grammer`（用于刷算法题时复习语言特性）
2. `2-computer-network`（从基础到 TCP、SSL 等深入）
3. `3-operating-system`（进程/内存管理与并发控制）
4. `4-database-mysql`（索引/执行流程/事务）
5. `5-ai-agent`（实践项目与扩展阅读）

## 贡献指南

欢迎贡献：新增笔记、修正错别字、补充图片或代码示例。贡献流程建议：

1. Fork 本仓库并在本地新建分支，分支命名示例：`fix/tcp-window-typo` 或 `feat/mysql-explain`。
2. 按项目已有风格提交 Markdown 文档与图片，图片放入对应目录的 `assets/` 下。
3. 在 PR 描述中说明修改点与理由，必要时附上渲染后的截图或示例。
4. 保持单次 PR 关注点单一（例如：仅修复一处笔误或添加一个专题），便于审阅。

文档格式建议：
- Markdown 使用简体中文书写。
- 图片文件名尽量使用英文或拼音并包含简短描述。
- 如添加代码片段，请确保代码能在常用环境下正确运行，并注明语言与运行方法。
